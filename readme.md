# ОР ППСП - Провинциальный район

Веб-приложение для сотрудников полиции.

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python app.py`

## Деплой на Render

1. Загрузите код на GitHub
2. Подключите репозиторий к Render
3. Render автоматически найдет render.yaml и развернет приложение
